# Wavy's World Boot Splash

## Description

Custom 3D boot splash screen featuring:
- **Background**: 3D black and purple space with animated stars
- **Character**: 3D animated character with:
  - Face tattoos/dermals
  - Love scars under right eye
  - Full arm, leg, and neck tattoos
  - Holding Glock with 30-round magazine
- **Text**: 
  - "Welcome to Wavy's World"
  - "Welcome to Glock's World Enjoy"

## Image Requirements

You'll need to create these image files (or use placeholders):

1. **character.png** - 3D character sprite (400x300px recommended)
   - Character walking animation
   - Face tattoos/dermals visible
   - Love scars under right eye
   - Full body tattoos (arms, legs, neck)
   - Holding Glock with 30-round magazine

2. **glock.png** - Glock pistol sprite (200x100px)
   - 30-round extended magazine visible

3. **text_welcome.png** - "Welcome to Wavy's World" text (600x100px)
   - Stylized 3D text

4. **text_glock.png** - "Welcome to Glock's World Enjoy" text (500x80px)
   - Stylized 3D text

5. **star.png** - Star sprite (20x20px)
   - For background animation

## Creating the Images

You can create these using:
- Blender (for 3D character)
- GIMP/Photoshop (for text and effects)
- Or use placeholder images initially

## Installation

The boot splash is automatically installed when creating the GhostPi image.

## Customization

Edit `wavys-world.script` to adjust:
- Animation speed
- Character position
- Text effects
- Star count and speed
