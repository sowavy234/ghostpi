# Flipper Buddy - AI-Powered Companion

**EDUCATIONAL PURPOSES ONLY**

Flipper Buddy is an AI-powered companion app for Flipper Zero that integrates with GhostPi's comprehensive pentesting toolkit.

## Features

- **Pentesting Tools Integration**: Access Kali, Parrot, and BlackArch tools
- **Brute Force Helper**: WiFi, SSH, PIN cracking tools
- **Marauder WiFi**: ESP32 Marauder board integration
- **AI Helper**: Code generation and attack planning assistance
- **GhostPi Connection**: Seamless integration with GhostPi system

## Building

```bash
# Install FBT first
fbt-build-helper.sh install

# Build Flipper Buddy
fbt-build-helper.sh build flipper_buddy
```

## Installation

1. Build the .fap file
2. Copy to Flipper Zero: `/apps/flipper_buddy/`
3. Launch from Flipper Zero apps menu

## Usage

- **Up/Down**: Navigate menu
- **OK**: Select/Execute
- **Back**: Exit
